# game-client

Phaser browser client for menus, lobby flow, HUD, and match rendering.

## Stack

- TypeScript
- Vite
- Phaser 3

## Run

From `apps/game-client`:

1. `npm install`
2. `npm run dev`

Vite will print the local URL (usually `http://localhost:5173`).

## Build

- `npm run build`
- `npm run preview`
